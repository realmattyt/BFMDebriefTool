Quick Start · iPad

BFM Debrief Tool

A 3D recreation of your engagement, built from ForeFlight track logs. Watch the merge, scrub the timeline, and study range & closure — right on your iPad.
Version 0.1  ·  Runs entirely on the device — nothing is uploaded.

1 Install Microsoft Edge (one time)

The tool is a single web page. iPad won’t run it in Safari from a file, but Microsoft Edge will.
  • Open your company app catalog (the managed app/“Company Portal” icon).
  • Find Microsoft Edge and install it.
Screenshot 01
Why Edge? It’s the one approved app that can both open the local file and let you pick your KML files. You only install it once.

2 Get the tool file

You’ll receive a single file named like bfm-debrief-v0.1.html.
  • Open the message/share I sent and save the file to your iPad (Files app).
  • The version number is in the filename — newer versions will be v0.2, v0.3, etc.
Screenshot 02

3 Open it in Edge

From the Files app, hand the file to Edge.
  1. In Files, press and hold bfm-debrief-v0.1.html.
  2. Tap Share.
  3. Choose Microsoft Edge (tap More first if you don’t see it).
Confirm these taps: this is the one device-specific step — please verify the exact path you used so it’s 100% right for everyone.
Screenshots 03–04

4 Try the demo first

See how it works before loading your own logs.
  • On the start screen, tap ▶ View demo flight.
  • The 3D replay loads, paused. Tap PLAY at the bottom to roll it.
Screenshot 05

5 Load your own flight

Export the track log from ForeFlight as KML for each aircraft, then load them here.
  1. On the start screen, tap the “Drop KML files here” box.
  2. Pick your .kml files from Files — normally two, one per jet.
  3. The replay builds automatically.
Tip: select both KML files at once so the two jets share one timeline.
